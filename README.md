# ConversorDec2Bin
##Esta es una guía para apreder a usar el conversor de decimal a binario.
1. El primer paso será abrir nuestro archivo dec2bin.py con el programa IDLE. Para ello abrimos IDLE, buscamos la opción Open y abrimos nuestro archivo.
2. Nos vamos a la opción "Run" y seleccionamos "Run module".
3. Ahora tan sólo tenemos que introducir primero el numero en decimal que queremos transformar y después la cantidad de dígitos que deseamos.
4. Listo. Ya tenemos convertido nuestro número a binario.
